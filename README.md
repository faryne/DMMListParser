## DMMListParser

### 編譯
`go build -o dmmlist`

### 使用方式
#### 每日上線 / 發售
```dmmlist -dmmurl https://www.dmm.co.jp/digital/videoa/-/delivery-list/=/delivery_date=2019-12-22/```

#### 根據品牌 
```dmmlist -dmmurl https://www.dmm.co.jp/digital/videoa/-/list/=/article=maker/id=45276/```
